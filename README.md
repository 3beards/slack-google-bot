SlackGoogleBot
==============

[![Build Status](https://travis-ci.org/dblock/slack-google-bot.png)](https://travis-ci.org/dblock/slack-google-bot)

Slack bot for Google, including CSE.

## Installation

See [DEPLOYMENT](DEPLOYMENT.md).

## Commands

### google hi

Politely says 'hi' back.

### google help

Get help.
